üîó URL Parameter Options

The following URL parameters are supported:

  • styleUrl - Load external CSS file
  • themeUrl - Alternate parameter name
  • cssUrl - Another alternate parameter name
  • theme - Built-in theme name (light, dark, high-contrast)
